Pair

is a 2-tuple, i.e. a datatype with two members.

<

defn

(Fn [(Pair a b), (Pair a b)] Bool) 

                    (< t1 t2)

=

defn

(Fn [(Pair a b), (Pair a b)] Bool) 

                    (= t1 t2)

>

defn

(Fn [(Pair a b), (Pair a b)] Bool) 

                    (> t1 t2)

a

instantiate

(Fn [(Ref (Pair a b) c)] (Ref a c))

gets the a property of a Pair.

b

instantiate

(Fn [(Ref (Pair a b) c)] (Ref b c))

gets the b property of a Pair.

copy

template

(Fn [(Ref (Pair a b) c)] (Pair a b))

copies the Pair.

delete

template

(Fn [(Pair a b)] ())

deletes a Pair. Should usually not be called manually.

                    (hash pair)

template

(Fn [a, b] (Pair a b))

creates a Pair.

(Fn [(Ref (Pair a b) c), a] ())

update-b

instantiate

updates the b property of a Pair using a function f.

zero

defn

(Fn [] (Pair a b)) 

                    (zero)

initializes a Pair by calling zero for all its members. zero must be defined for all member types.
