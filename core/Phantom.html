Phantom

is a type without inhabitants and a free type variable, which is a fancy way of saying that it has no runtime representation.

When trying to mark types programmatically using something like the but without the need for a special form, Phantom comes in handy.

Example:

(defn test [a]
  (Phantom.ensure a (the (Phantom Int) (Phantom.init))))

(println* (test 123l)) ; this will not compile

copy

template

(Fn [(Ref (Phantom a) b)] (Phantom a))

copies the Phantom.

delete

template

(Fn [(Phantom a)] ())

deletes a Phantom. Should usually not be called manually.

ensure

defn

(Fn [a, (Phantom a)] a) 

                    (ensure a p)

is a function that takes anything and a Phantom, ensuring that the type contained by the Phantom and the type of the first parameter match.

init

template

(Fn [] (Phantom a))

creates a Phantom.

(Fn [(Ref (Phantom a) b)] String)
